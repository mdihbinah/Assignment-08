# 🧱 Tiles Gallery

## 🔗 Live URL
https://assignment-08-iota.vercel.app/

## 📌 Project Purpose
A tile showcase platform where users can explore, search, and manage tile collections with authentication.

## ✨ Key Features
*   **Authentication:** Secure Login and Registration powered by **Better Auth**, including **Google Social Login**.
*   **Dynamic Gallery:** A searchable "All Tiles" page with real-time search by tile title.
*   **User Profile:** A dedicated private route for users to view their data and update their **Name** and **Profile Image**.
*   **Interactive UI:** 
    *   **Marquee:** Scrolling announcements for new arrivals.
    *   **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.
*   **Data Fetching:** Robust integration with a JSON server/API to manage tile details.   
*   **Loading States:** Custom loaders implemented for a smooth user experience during data fetching.

---

## 🛠️ Technologies Used
- Next.js (App Router)
- MongoDB
- BetterAuth
- DaisyUI
- JSON Server
